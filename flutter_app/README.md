# Panorama (Flutter)

Desktop file explorer port of the Electron/React Panorama app.

## Run

```bash
flutter pub get
flutter run -d macos   # or windows / linux
```

## Build

```bash
flutter build macos
flutter build windows
flutter build linux
```

## Notes

Writes to `../notes/improvements.json` (repo root) so the Cursor improvement-notes skill keeps working.
